# Meteogram Widget

A mobile weather widget showing temperature, precipitation, and cloud cover as a meteogram.

## Features

- 48-hour forecast with 2 hours of recent history
- Temperature line, precipitation bars, cloud cover background
- Home screen widget (Android & iOS)
- Light/dark theme support
- Multi-language (EN, DE, FR, ES, IT)

## Quick Start

```bash
flutter pub get
flutter gen-l10n
flutter run
```
